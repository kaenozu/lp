# ブランド名候補と分析

2026-07-18 時点の初期評価。

## 背景

現在の「小さなアプリ工房」は、以下と類似・競合している:
- 「東京・小さなアプリ工房」（合同会社プライム東京）
- 「れもんの小さなアプリ工房」（別個人開発者）
- Google Play に「アプリ工房」を含むデベロッパー名が複数存在

→ 識別性が弱いため、固有ブランド名への移行を推奨。

---

## 候補一覧

### 1位: アプノキ（apunoki）◎

- **読み**: あぷのき
- **由来**: 「アプリ」+「木」。アプリを一つずつ育て、枝葉を増やすイメージ
- **良い点**: 4文字で短い、造語性が高く検索占有しやすい、ロゴ展開しやすい、ジャンルを限定しない
- **気になる点**: 初見で意味が伝わらない、開始直後は補足説明が必要
- **ドメイン案**: apunoki.jp / apunoki.dev / apunoki.app / apunoki.com
- **SNS**: @apunoki 空き可能性: 高

### 2位: アプ日和（apubiyori）◎寄りの○

- **読み**: あぷびより
- **由来**: 「アプリ」+「日和」。日常に自然なじむアプリ
- **良い点**: 柔らかい、親しみやすい、SNS投稿に使いやすい
- **気になる点**: 誤読リスク、「○○日和」は汎用表現で造語強度は低め
- **ドメイン案**: apubiyori.jp / apubiyori.dev / apubiyori.app / apubiyori.com
- **SNS**: @apubiyori 空き可能性: 高

### 3位: こまどラボ（komadolab）○

- **読み**: こまどらぼ
- **由来**: 「小窓」+「ラボ」。日常の困りごとに小さな窓を開ける道具を作る場所
- **良い点**: 発音しやすい、個人開発の立ち位置を表現、「工房」より現代的
- **気になる点**: 「小窓」は住宅・建築を連想、技術寄りの印象
- **ドメイン案**: komadolab.jp / komadolab.dev / komadolab.com
- **SNS**: @komadolab 空き可能性: 高

### 4位: まめアプ（mameapp）○

- **読み**: まめあぷ
- **由来**: 「小さい」+「こまめ・誠実」。丁寧に作ったアプリ
- **良い点**: 分かりやすい、親しみやすい、アイコン展開しやすい
- **気になる点**: 「まめ」を含む既存サービス多、「まめパル」など類似あり
- **ドメイン案**: mameapp.jp / mameapp.dev / mameapps.jp
- **SNS**: @mameapp 空き可能性: 中

### 5位: まめ灯（mameakari）○

- **読み**: まめあかり
- **由来**: 暮らしの小さな不便をそっと照らす
- **良い点**: 最も温かみが強い、誠実な姿勢を表現
- **気になる点**: 誤読リスク、アプリ開発と分かりにくい、「まめ電」など類似あり
- **ドメイン案**: mameakari.jp / mameakari.dev / mameakari.app
- **SNS**: @mameakari 空き可能性: 中

## 検索・商標チェック手順

### Google検索（例: アプノキ）

```
"アプノキ"
"アプノキ" アプリ
"アプノキ" Android
"アプノキ" 開発
"アプノキ" サービス
"アプノキ" 会社
"あぷのき"
"APUNOKI"
"APU NOKI"
site:play.google.com "アプノキ"
site:x.com "アプノキ"
site:instagram.com "アプノキ"
```

確認観点:
- 同じ読み方の別表記
- 一文字違いのサービス
- 同業・隣接分野で使われている名前
- 活動停止中だが検索結果に残っているブランド
- Google Playのデベロッパー名
- X, Instagram, YouTube, GitHub, note

### ドメイン確認

Value Domain / Cloudflare Registrar / お名前.com で以下を確認:
- .jp / .dev / .app / .com が空いているか
- 2年目以降の更新価格
- 第三者取得済みか
- 中古ドメイン（過去利用履歴）か

優先順位: .jp > .com > .app > .dev

### J-PlatPat 商標簡易検索手順

1. J-PlatPat を開く
2. 上部メニュー「商標」→「商標検索」
3. 検索対象を「出願・登録情報」に
4. 「商標（検索用）」で表記を検索
5. 「称呼（類似検索）」で読み方を全角カタカナで検索

例: アプノキの場合
- 商標（検索用）: アプノキ / あぷのき / APUNOKI
- 称呼（類似検索）: アプノキ / アプリノキ / アップノキ

注意: 公開後タイムラグあり。ゼロでも絶対に存在しないとは断定できない。

## 推奨

**「アプノキ」を最優先で検討。**

理由:
1. アプリとの関連性がありながら一般名詞ではない
2. 4文字で覚えやすく統一しやすい
3. 将来のどのジャンルにも使える
4. 「一つずつ育てる」「暮らしに根づく」という個人開発の温かさを表現できる

### 正式表記案

```
アプノキ
小さなアプリを、ひとつずつ。
```

サイトタイトル（移行期間）:
```
アプノキ｜個人開発の小さなアプリ
```

### 「小さなアプリ工房」を使い続ける場合の対策

固有名詞としては使わず、説明文に降格:
```
アプノキ｜小さなアプリ工房
```

## 確定後のアクション

**1日目（候補確定後即日）**:
- 対象候補のGoogle完全一致検索
- Google Play / X / Instagram / YouTube / GitHub / note 検索
- J-PlatPat 表記・称呼両方検索
- .jp / .dev / .app / .com 同時確認
- 問題なければ即日確保: ドメイン, X, Instagram, GitHub組織名, Google Play デベロッパー表記

**その後**:
- ポータルサイト名・OGP・構造化データ更新
- ロゴとファビコン作成
- Google Play デベロッパー情報更新
- 各アプリの「開発者について」
- プライバシーポリシー・利用規約・問い合わせ先更新
- GitHub プロフィール・README更新
- 旧サイト名から新ブランドへの案内追加

**商標出願のタイミング**: 複数アプリを長期運用、広告費投入、課金アプリ展開の段階で検討。
